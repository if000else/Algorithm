# 动态规划

- 背包问题的最优解算法
- 适用于需要再给定约束条件下优化某种指标
- 适用于问题可分解为离散子问题的场景
- 需要使用网格的办法，单元格中的值就是需要优化的值，右下角的值为最优解
- 动态规划解决方案并不是任何时候都准，但是可以用来预测
